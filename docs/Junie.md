# 🧠 JUNIE.MD — GEDÄCHTNISPROTOKOLL (PROJECT MEMORY)

## 📌 GESAMTPROJEKT: JETBRAIN — CORONA CONTROL ULTIMATE (CLOUD-RUN-ONLY)
**JetBRAIN** ist eine photorealistische 3D-Polizei-Simulation der nächsten Generation im Browser. 
- **Thema:** Crowd-Management am Stephansplatz, Wien (17. März 2021).
- **Technik:** React 19, Three.js (WebGPU Fokus), Zustand, Web Worker.
- **Hardware-Vorgabe:** ZERO Local GPU/CPU/Festplatte. Alles läuft über die Cloud (Zero-Footprint).
- **Browser:** Brave Nightly (`C:\Program Files\BraveSoftware\Brave-Browser-Nightly\Application>brave.exe`).
- **Clouds:** GitHub, Huggingface, GitLab, GitKraken.
- **Einzigartigkeit:** Hyper-AAA Grafik (200k+ Poly NPCs), 24h-Echtzeit-Event-System (24 Min = 24 Std).

## 🎯 ZIEL DER ENTWICKLUNG
Erstellung eines voll funktionsfähigen, taktischen 3D-Web-Games mit maximaler Grafikleistung, autonomer NPC-KI (Tension System) und einer 100%igen Validierung durch automatisierte Test-Suiten.
